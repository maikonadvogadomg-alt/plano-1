# Como continuar sem terminal

## Situação atual
O caminho mais seguro é não depender do terminal daqui.

## Fluxo recomendado

### Passo 1
Aplicar os arquivos no editor.

### Passo 2
Enviar o projeto para o GitHub pelo botão 🌿.

### Passo 3
Abrir o projeto no StackBlitz.

### Passo 4
Executar no terminal de lá o que for necessário.
