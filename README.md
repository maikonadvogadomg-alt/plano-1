# Meu Projeto

Projeto web simples em HTML, CSS e JavaScript puro.

Em linguagem simples: hoje este projeto é uma página estática no navegador, com visual básico e um pequeno script carregado ao abrir a página.

---

## Status atual

Atualmente, o projeto possui:

- interface em `HTML`
- estilos em `CSS`
- comportamento básico em `JavaScript`
- documentação da API planejada
- documentação interna para retomada do projeto depois
- registro de continuidade sem terminal

Atualmente, o projeto **não possui**:

- backend real
- rotas `/api` funcionando
- banco de dados
- autenticação
- integração real com IA no servidor
- dependências externas instaladas por npm
