# Perfil de aprendizado da Jasmim

## Sobre Saulo
- Saulo é advogado.
- Saulo tem limitação nos membros superiores.
- Saulo usa bastante comando de voz.
- Saulo prefere linguagem simples, direta e sem excesso de tecnicês.
- Quando o assunto é técnico, funciona melhor explicar com analogias simples.

## Preferências de resposta
- Respostas curtas e claras funcionam melhor.
- É importante sempre deixar um próximo passo concreto.
- Documentação precisa ser honesta, sem inventar recurso que ainda não existe.

## Sobre este projeto
- Neste momento o projeto é um site estático simples.
- Arquivos atuais principais: `index.html`, `style.css`, `script.js`.
- Ainda não existe backend real nem rotas de API implementadas.
- Há interesse em temas como PWA, APK, servidor, chave de API e independência de plataforma.
