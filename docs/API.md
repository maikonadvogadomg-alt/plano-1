# Documentação da API

## Visão geral
No estado atual, este projeto **não possui uma API backend implementada**.

Em linguagem simples: hoje o projeto é só a “parte da tela”.  
Ainda não existe o “balcão de atendimento” no servidor para receber pedidos por rota como `/api/...`.

Mesmo assim, este documento organiza a **estrutura recomendada da futura API**, para servir como base do projeto.
